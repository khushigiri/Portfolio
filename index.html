Task Management

Built with HTML, CSS, JavaScript, and Bootstrap, this Task Management Application enables users to create, view, edit, and delete tasks with ease.

The user-friendly interface ensures smooth navigation and an intuitive experience across devices, thanks to responsive design elements provided by Bootstrap.

Book API

This project is a RESTful API built with Node.js, Express.js, and MongoDB for managing books and user data in a library or book record system.

The API supports full CRUD operations (Create, Read, Update, Delete) for both entities and is designed to be scalable, secure, and easy to integrate with frontend applications or third-party services.

BookMyShow Clone

I created a BookMyShow clone using React, Tailwind CSS, and the TMDB API to simulate an online movie booking platform.

The project features a clean, responsive UI with dynamic routing and reusable components, allowing users to explore movies seamlessly across devices.

React was used for managing components and state, while Tailwind CSS handled the layout and styling.

The TMDB API integration enables real-time fetching of movie data, including posters, ratings, genres, and descriptions. Key features include a trending movie carousel, genre-based filtering, and detailed movie pages.
